# Проект Python - Установка среды окружения

Этот проект предоставляет средства для установки и запуска выделенной среды окружения для python с использованием Docker.

## Структура репозитория

- `Dockerfile`: конфигурация для создания Docker-контейнера с установленным python и всеми необходимыми зависимостями.
- `start.sh`: скрипт для старта проекта, который запускает необходимую среду и выполняет проект.

## Установка

1. Клонирование репозитория

Склонируйте этот репозиторий на вашу локальную машину:

```
git clone https://github.com/ваш_проект.git
cd ваш_проект
```

2. Построение Docker-образа

Соберите Docker-образ.

3. Запуск контейнера.

После того как образ будет построен, вы можете запустить контейнер.
Контейнер запустит скрипт start.sh, который создаст виртуальное окружение и запустит проект.
